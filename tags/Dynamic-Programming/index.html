题解【洛谷 P6021 -- 洪水】

给定一棵 $n$ 个点的有根树，点有点权。$q$ 次操作，每次操作修改一个点的权值，或者给一个点并只考虑以该点为根的最大子树，在这个子树中删去一些点使得每个叶子结点都与根不连通，删去的点的权值和的最小值。

$n,m\le2\times10^5$，点权始终不超过 $2\times10^9$。
